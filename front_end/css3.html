CSS3简介

CSS 用于控制网页的样式和布局。

CSS3 是最新的 CSS 标准。

同时，css3中加入了许多新属性，如border-radius(圆角)，gradients(渐变)，transition(过渡)，@keyframes+animation(动画)。

左侧是一个弹簧动画特效，就是用css3做出来的哦。
